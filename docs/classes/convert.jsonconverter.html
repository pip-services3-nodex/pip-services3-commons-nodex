Class JsonConverter

Converts arbitrary values from and to JSON (JavaScript Object Notation) strings.

Example

let value1 = JsonConverter.fromJson("{\"key\":123}"); // Result: { key: 123 }
let value2 = JsonConverter.toMap({ key: 123}); // Result: "{\"key\":123}"
